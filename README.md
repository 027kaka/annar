# Anna Remax UI

一款基于 `Remax` 框架开发的小程序 UI 组件库

![Image text](https://smebimage.fuliaoyi.com/FjMomAWXPUzaSAcAkLzr920trN-p)

## 特性

- 基于 `Remax` 开发 UI 组件
- 一套组件可以在 `支付宝小程序`，`钉钉小程序` 两端适配运行（后续会增加对微信小程序的支持）

## 安装

npm install anna-remax-ui

## 使用

在代码中 `import` 需要的组件

```js
import { Button } from 'anna-remax-ui'
```
